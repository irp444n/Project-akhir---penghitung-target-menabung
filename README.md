Project akhir yang berjudul “Penghitung Target Menabung” adalah program sederhana berbasis teks yang dibuat menggunakan bahasa C di Visual Studio Code (VS Code). Tujuan dari program ini adalah membantu pengguna menghitung perkiraan waktu yang dibutuhkan agar tabungan mencapai target yang diinginkan. Pengguna diwajibkan untuk memasukkan nama atau deskripsi dari target tabungan, beserta harga target, jumlah tabungan awal, dan jumlah uang yang akan ditabung setiap bulannya. Program ini menggunakan perulangan while untuk menambahkan jumlah tabungan setiap bulan secara otomatis hingga mencapai atau melebihi nilai target, serta menampilkan persentase keberhasilan setiap bulannya. Setelah target tercapai, program akan menampilkan jumlah bulan yang dibutuhkan untuk mencapai target tersebut. Proyek ini memperlihatkan penerapan konsep dasar pemrograman seperti input dan output, operasi aritmatika, serta perulangan. Secara keseluruhan, proyek ini tidak hanya membantu pengguna merencanakan keuangan, tetapi juga menjadi latihan praktis dalam memahami logika pemrograman dan struktur kontrol dalam bahasa C.
